# Gym Tracker

A personal gym tracking web app built with Python and Streamlit. Designed to help you log workouts, track PRs, time rest periods, and stay consistent.

## What it does

- Log your exercises, weight, sets, and reps for each session
- Automatically detects and highlights new personal records
- Tracks your gym streak across days
- Shows your best lifts for the week on the dashboard
- Rest timer with countdown and progress bar
- 1RM calculator using the Epley formula
- Custom split selector so you can set your own training days
- Warmup checklist to make sure you prep before lifting
- Exercise database with autocomplete — add and remove exercises
- Saves all your data locally so nothing gets lost between sessions

## Built with

- Python
- Streamlit
- Pandas
- JSON (for local data storage)
